Unnamed: 0              - Row number, can be ignored
Subject                 - Unique participant ID number
Rest.Scan               - Which resting state scan the row refers to (e.g. rest_1)
MeanFD                  - Average head movement per frame in mm, main data quality metric
NumFD_greater_than_0.20 - Number of frames where head movement exceeded 0.20mm
rootMeanSquareFD        - Root mean square of framewise displacement, more sensitive to movement spikes
FDquartile.top1.4thFD.  - Average movement in the top 25% most-motion frames
PercentFD_greater_than_0.20 - Percentage of total frames with movement over 0.20mm
MeanDVARS               - Measures signal intensity changes between frames, another motion artifact indicator
MeanFD_Jenkinson        - Framewise displacement calculated using the Jenkinson formula, alternative to MeanFD
site                    - Research site where participant was scanned (KKI, NYU, OHSU, Peking_1/2/3)
sibling_id              - ID of a sibling also in the study, mostly empty
data_set                - Whether participant is in training set or test set
age                     - Participant age in years
sex                     - Participant sex (M/F)
handedness              - Dominant hand (R/L/ambidextrous)
